Fonctionalités du script

Le script permet de pratiquer de façon automatisée et sécurisée des échanges de liens immédiat avec n'importe quel site. Le script a également été fait de façon a optimiser votre référencement sur des mots-clés (ou expressions) que vous pouvez librement choisir. Vous pouvez en personnaliser l'aspect facilement en modifiant la feuille de styles.
Chaque webmaster peut inscrire son site via le formulaire disponible. Si le webmaster a au préalable mis en place un lien vers votre site, son site est accepté sinon il est refusé. Le script possède une interface d'administration ainsi que la possibilité de faire différents réglages sans qu'aucune connaissances particulières soient nécessaires.

Les principales fonctionalités :

  • Echange de liens en dur ou non (au choix).
  • Filtrage d'URL par inscription sur liste noire.
  • Autoriser ou non les URL avec paramètres.
  • Autoriser ou non les URL vers les pages internes d'un site.
  • Doublon d'inscription interdit.
  • Script de vérification de la réciprocité des échanges de liens.
  • Possibilité d'envoyer un email pour informer de la validité de l'échange de lien
  • etc.

Minimum requis

Hébergement :

  • PHP 4 et +.
  • Accès FTP à l'hébergement de votre site.

Connaissances :

  • Savoir éditer un fichier php (aucune connaissance en php n'est requise).
  • Savoir mettre en ligne des fichiers via FTP.
  • Savoir modifier les autorisations d'accès à un fichier via FTP (voir ci-dessous) ou SSH (chmod).

Logiciels

  • Bloc Note de Windows ou éditeur HTML/PHP pour modifier la configuration du script (1 seule fois).
  • Logiciel client FTP permettant le changement des autorisations d'accès (FileZilla en téléchargemement gratuit)

Installation / Configuration

  • Dézippez le contenu de l'archive autolien_v5.zip dans un dossier temporaire.
  • Coupez-collez le dossier partenaires dans le dossier local de votre site, si possible au même niveau que votre page d'accueil.
  • Editez le fichier settings.php pour faire les réglages suivants entre "DEBUT DE LA CONFIGURATION" et "FIN DE LA CONFIGURATION" :
    • Choix de votre type d'hébergement : 1 = UNIX (Linux), 2 = Windows, 3 = Mac.
    • Mot de passe pour accéder à l'administration (veuillez choisir un mot de passe assez complexe pour sécuriser le script, au minimum 10 caractères alphanumériques).
    • Le nom de votre site.
    • L'URL de votre site sans / à la fin !
    • Le répertoire d'installation du script avec un / à la fin !
    • La liste des expressions clés (mots-clés) avec lesquels les liens réciproques seront faits vers votre site.
    • Votre adresse email (il est INDISPENSABLE de mettre une adresse valide).
    • Envoi d'un email au partenaire ayant ajouté son site : 1 = oui, 0 = non.
    • Nombre maximum de liens par page (20 maximum conseillé).
    • Nombre des derniers sites ajoutés à mettre en valeur (5 maximum conseillé) en première page.
    • Les URL ajoutés doivent-être celle de la page d'accueil du site partenaire : 1 = oui, 0 = non.
    • Les URL ajoutés peuvent avoir des paramètres : 1 = oui, 0 = non.
    • Autoriser les inscriptions multiples par domaine (concerne surtout les hébergeurs gratuits) (ex. toto.free.fr, titi.free.fr, ...) ? 1 = oui, 0 = non. ATTENTION: en mettant oui, un webmaster pourrait faire un échange de liens sur toutes ses pages pour le même site.
    • Utiliser la liste noire pour filtrer les inscriptions ? 1 = oui, 0 = non
    • Qualité des liens, lien en dur
    • Où placer les nouveaux liens ajoutés par l'administration ? 0 = en début de liste, 1 = en fin de liste
  • Enregistrez le fichier settings.php.
  • Mettez en ligne par FTP le dossier partenaires et l'ensemble de son contenu.
  • Faites un chmod 666 sur dataliens1.txt (autorise l'écriture / lecture pour tous), ou utilisez la fonction de FileZilla Attribut du fichier (clique droit sur le fichier distant).
  • Faites un chmod 666 sur siterefuse.txt (autorise l'écriture / lecture pour tous), ou utilisez la fonction de FileZilla Attribut du fichier (clique droit sur le fichier distant).

Administration

Pour accéder à votre page d'échange de liens : http://www.domaine.com/partenaires/index.php (à personnaliser selon votre configuration)
Pour accéder à l'administration du script : http://www.domaine.com/partenaires/admin.php (à personnaliser selon votre configuration)

Les fonctions de l'administration :

  • Supprimer un lien.
  • Voir la page où se trouve votre lien réciproque sur les sites inscrits.
  • Vérifiez la réciprocité des échanges de liens (les liens brisés ou inexistant entraînent la suppression automatique du site concerné).
  • Inscrire un URL dans la liste noire (blacklisté).
  • Ajouter un lien, avec ou sans lien réciproque obligatoire (pour vos propres sites par exemple).

Personnalisation

Vous pouvez personnaliser l'apparence du script en modifiant la feuille de styles style.css. Vous pouvez également modifier les fichiers header.php et footer.php pour les personnaliser en fonction de vos besoins. Il vous est recommandé d'agir avec précaution afin de ne pas altérer le fonctionnement du script.
